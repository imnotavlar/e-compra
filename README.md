# 🛒 E-Compra

O **E-Compra** é um sistema desktop desenvolvido em **C#** utilizando **.NET**, criado como projeto das disciplinas de **Programação Orientada a Objetos** e **Estrutura de Dados** da Fatec Cruzeiro.

O objetivo do sistema é facilitar o gerenciamento e a compra de cestas básicas, oferecendo uma interface simples e funcionalidades essenciais para o controle dos pedidos.

## ✨ Funcionalidades

* 📦 Cadastro de cestas básicas;
* 🛍️ Cadastro de produtos;
* 🛒 Realização de compras;
* 📊 Dashboard com atualização automática conforme os pedidos são realizados;
* 📄 Geração de PDF com os detalhes do pedido.

## 🚀 Tecnologias Utilizadas

* C#
* .NET
* Windows Presentation Foundation (WPF)
* Programação Orientada a Objetos (POO)
* Estrutura de Dados

## 🎯 Desafio do Projeto

Um dos principais desafios deste projeto foi desenvolver toda a aplicação **sem utilizar um banco de dados externo**.

Para isso, as informações foram gerenciadas utilizando estruturas de dados implementadas durante a disciplina, colocando em prática conceitos fundamentais de organização, armazenamento e manipulação de dados.
